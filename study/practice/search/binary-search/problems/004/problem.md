# 問題 004 — K 分割の最小の最大和

長さ $N$ の整数列 $A = (A_1, A_2, \ldots, A_N)$ と整数 $K$ が与えられます。

$A$ を左から順に、空でない $K$ 個の連続部分列にすべて分割するとき、各部分列の和の最大値としてあり得る最小値を求めてください。

## 制約

- $1 \leq K \leq N \leq 10^5$
- $1 \leq A_i \leq 10^9$
- 入力はすべて整数

## 入力

入力は以下の形式で標準入力から与えられる。

```
N K
A_1 A_2 ... A_N
```

## 出力

各部分列の和の最大値としてあり得る最小値を1行で出力せよ。

## 入力例

```
5 3
2 4 4 5 9
```

## 出力例

```
9
```

分割 `[2, 4]`, `[4, 5]`, `[9]` では各部分列の和は 6, 9, 9 であり、最大値は 9 である。最大和を 8 以下にすることはできない。
