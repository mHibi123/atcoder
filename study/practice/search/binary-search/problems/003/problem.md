# 問題 003 — 値 X の個数

長さ $N$ の昇順にソートされた整数列 $A = (A_1, A_2, \ldots, A_N)$ が与えられます。

$Q$ 個のクエリが与えられます。$i$ 番目のクエリでは整数 $X_i$ が与えられるので、$A$ の中で $X_i$ と等しい要素の個数を出力してください。

## 制約

- $1 \leq N, Q \leq 2 \times 10^5$
- $1 \leq A_1 \leq A_2 \leq \cdots \leq A_N \leq 10^9$
- $1 \leq X_i \leq 10^9$
- 入力はすべて整数

## 入力

入力は以下の形式で標準入力から与えられる。

```
N
A_1 A_2 ... A_N
Q
X_1
X_2
...
X_Q
```

## 出力

$Q$ 行出力せよ。$i$ 行目には $i$ 番目のクエリの答えを出力せよ。

## 入力例

```
5
1 3 3 5 7
3
3
5
2
```

## 出力例

```
2
1
0
```

$X=3$ は 2 個、$X=5$ は 1 個、$X=2$ は 0 個である。
