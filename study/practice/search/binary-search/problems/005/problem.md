# 問題 005 — K 個選ぶ最小差の最大化

長さ $N$ の昇順にソートされた相異なる整数列 $A = (A_1, A_2, \ldots, A_N)$ と整数 $K$ が与えられます。

$A$ から $K$ 個の要素を選び、選んだ要素を昇順に並べたときの隣接する2要素の差の最小値としてあり得る最大値を求めてください。

## 制約

- $1 \leq K \leq N \leq 10^5$
- $1 \leq A_1 < A_2 < \cdots < A_N \leq 10^9$
- 入力はすべて整数

## 入力

入力は以下の形式で標準入力から与えられる。

```
N K
A_1 A_2 ... A_N
```

## 出力

隣接する2要素の差の最小値としてあり得る最大値を1行で出力せよ。

## 入力例

```
6 3
1 2 3 5 9 10
```

## 出力例

```
4
```

$1, 5, 9$ を選ぶと差は 4, 4 であり、最小値は 4 である。最小差を 5 以上にすることはできない。
