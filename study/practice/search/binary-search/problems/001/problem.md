# 問題 001 — ソート済み配列の存在判定

長さ $N$ の昇順にソートされた整数列 $A = (A_1, A_2, \ldots, A_N)$ が与えられます。

$Q$ 個のクエリが与えられます。$i$ 番目のクエリでは整数 $X_i$ が与えられるので、$A$ に $X_i$ と等しい要素が存在するなら `Yes`、存在しないなら `No` を出力してください。

## 制約

- $1 \leq N, Q \leq 2 \times 10^5$
- $1 \leq A_1 \leq A_2 \leq \cdots \leq A_N \leq 10^9$
- $1 \leq X_i \leq 10^9$
- 入力はすべて整数

## 入力

入力は以下の形式で標準入力から与えられる。

```
N
A_1 A_2 ... A_N
Q
X_1
X_2
...
X_Q
```

## 出力

$Q$ 行出力せよ。$i$ 行目には $i$ 番目のクエリの答えを出力せよ。

## 入力例

```
5
1 3 5 7 9
3
5
4
9
```

## 出力例

```
Yes
No
Yes
```
