# 問題 003 — 矩形区域の和

$H$ 行 $W$ 列の整数が書かれたグリッドが与えられます。上から $i$ 行目、左から $j$ 列目の値を $A_{i,j}$ とする。

$Q$ 個のクエリが与えられます。$i$ 番目のクエリでは $(R_1, C_1, R_2, C_2)$ が与えられるので、行 $R_1$ から $R_2$、列 $C_1$ から $C_2$ の矩形区域に含まれる整数の和を出力してください。

## 制約

- $1 \leq H, W, Q \leq 500$
- $-10^9 \leq A_{i,j} \leq 10^9$
- $1 \leq R_1 \leq R_2 \leq H$
- $1 \leq C_1 \leq C_2 \leq W$
- 入力はすべて整数

## 入力

入力は以下の形式で標準入力から与えられる。

```
H W
A_{1,1} A_{1,2} ... A_{1,W}
A_{2,1} A_{2,2} ... A_{2,W}
...
A_{H,1} A_{H,2} ... A_{H,W}
Q
R_1 C_1 R_2 C_2
R_1 C_1 R_2 C_2
...
```

## 出力

$Q$ 行出力せよ。$i$ 行目には $i$ 番目のクエリの答えを出力せよ。

## 入力例

```
3 3
1 2 3
4 5 6
7 8 9
2
1 1 2 2
2 2 3 3
```

## 出力例

```
12
28
```

1 番目のクエリは左上 2×2 区域（$1+2+4+5=12$）、2 番目のクエリは右下 2×2 区域（$5+6+8+9=28$）である。
