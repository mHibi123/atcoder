# 問題 002 — 和が K になる連続部分列の個数

長さ $N$ の整数列 $A = (A_1, A_2, \ldots, A_N)$ と整数 $K$ が与えられます。

連続する部分列（空でない）のうち、要素の和がちょうど $K$ になるものの個数を求めてください。

## 制約

- $1 \leq N \leq 2 \times 10^5$
- $-10^9 \leq A_i \leq 10^9$
- $-10^{14} \leq K \leq 10^{14}$
- 入力はすべて整数

## 入力

入力は以下の形式で標準入力から与えられる。

```
N
A_1 A_2 ... A_N
K
```

## 出力

和が $K$ になる連続部分列の個数を1行で出力せよ。

## 入力例

```
5
1 2 3 -3 1
3
```

## 出力例

```
3
```

和が $3$ になる連続部分列は `[3]`, `[1, 2]`, `[2, 3, -3, 1]` の3つである。
