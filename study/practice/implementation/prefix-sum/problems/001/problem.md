# 問題 001 — 区間和クエリ

長さ $N$ の整数列 $A = (A_1, A_2, \ldots, A_N)$ が与えられます。

$Q$ 個のクエリが与えられます。$i$ 番目のクエリでは整数 $L_i, R_i$ が与えられるので、$A_{L_i} + A_{L_i+1} + \cdots + A_{R_i}$ を出力してください。

## 制約

- $1 \leq N, Q \leq 2 \times 10^5$
- $-10^9 \leq A_i \leq 10^9$
- $1 \leq L_i \leq R_i \leq N$
- 入力はすべて整数

## 入力

入力は以下の形式で標準入力から与えられる。

```
N
A_1 A_2 ... A_N
Q
L_1 R_1
L_2 R_2
...
L_Q R_Q
```

## 出力

$Q$ 行出力せよ。$i$ 行目には $i$ 番目のクエリの答えを出力せよ。

## 入力例

```
5
1 2 3 4 5
3
1 3
2 4
5 5
```

## 出力例

```
6
9
5
```
