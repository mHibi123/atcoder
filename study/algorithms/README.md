## 練習フォルダの命名

練習を始めるときは `_template` を `practice/{category_slug}/{algorithm_slug}/` にコピーしてください。

| category_slug     | 対応ファイル         |
| ----------------- | -------------------- |
| `implementation`  | implementation.md    |
| `search`          | search.md            |
| `greedy`          | greedy.md            |
| `dp`              | dp.md                |
| `data-structure`  | data-structure.md    |
| `math`            | math.md              |
| `graph`           | graph.md             |
| `string`          | string.md            |

各アルゴリズムの slug はカテゴリ md 内の HTML コメントを参照してください。
